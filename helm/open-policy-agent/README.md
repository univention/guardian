# open-policy-agent

OPA is a full-featured policy engine that offloads policy decisions from your service.

- **Version**: 0.1.0
- **Type**: application
- **AppVersion**: 0.53.0
-

## Introduction

This chart does install the the Open Policy Agent component for Guardian.
